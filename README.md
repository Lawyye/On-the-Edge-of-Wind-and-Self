# 🧠 Psychology Bot - Анонимная психологическая поддержка

Telegram бот для анонимного общения и получения психологической поддержки от сообщества.

## Возможности

- 🆘 Оставить анонимную заявку о проблеме
- 💪 Помочь другому пользователю
- 📝 Отвечать на заявки
- 🤐 Полная анонимность
- 💬 Опциональный контакт после общения

## Установка на AlwaysData

### Шаг 1: Создать аккаунт на AlwaysData
1. Перейди на https://www.alwaysdata.com/
2. Создай бесплатный аккаунт
3. Выбери Python 3.10+

### Шаг 2: Загрузить проект
```bash
git clone https://github.com/Lawyye/On-the-Edge-of-Wind-and-Self
cd On-the-Edge-of-Wind-and-Self
```

### Шаг 3: Установить зависимости
На AlwaysData в контроле:
1. Перейди в "Environment → Python packages"
2. Установи:
   - python-telegram-bot
   - psycopg2-binary
   - python-dotenv

### Шаг 4: Настроить БД
1. На AlwaysData создай PostgreSQL базу
2. Скопируй строку подключения
3. Переименуй `.env.example` в `.env`:
```bash
mv .env.example .env
```
4. Открой `.env` в редакторе и заполни:
```
TELEGRAM_TOKEN=твой_токен_бота
DATABASE_URL=postgresql://пользователь:пароль@хост/база
```

### Шаг 5: Запустить бот
На AlwaysData в консоли:
```bash
python bot.py
```

## Как получить токен Telegram бота

1. Напиши @BotFather в Telegram
2. Используй `/newbot`
3. Следуй инструкциям
4. Скопируй токен в `.env`

## Как это работает

### Для человека, нуждающегося в помощи:
1. Напиши `/start` боту
2. Выбери "🆘 Мне нужна помощь"
3. Заполни форму (название, описание, категория)
4. Получи ID заявки
5. Жди ответов от других людей

### Для человека, помогающего:
1. Напиши `/start` боту
2. Выбери "💪 Помочь другому"
3. Смотри список открытых заявок
4. Выбери заявку и напиши ответ
5. Если твой ответ понравится - получишь контакт
